# ⚡ 快速开始指南

## 🎯 三步完成设置

### 第 1 步：运行数据库脚本 (5分钟)

1. 打开 https://app.supabase.com
2. 选择你的项目
3. 点击 **SQL Editor** → **New Query**
4. 复制 `supabase-setup.sql` 的内容
5. 粘贴并点击 **Run**

### 第 2 步：启用邮箱认证 (2分钟)

1. 点击 **Authentication** → **Providers**
2. 确保 **Email** 已启用 ✅
3. 点击 **Authentication** → **URL Configuration**
4. 设置 **Site URL** 为 `http://localhost:5173`

### 第 3 步：测试功能 (3分钟)

```bash
# 启动开发服务器
npm run dev
```

1. 打开 http://localhost:5173
2. 点击右上角"登录"按钮
3. 注册一个新账户
4. 登录测试

## 🆘 遇到问题？

### 常见问题快速解决：

**Q: 注册后无法登录？**
A: 检查是否启用了邮箱确认。如果启用了，需要点击邮件中的确认链接。

**Q: 没有收到邮件？**
A: 
- 检查垃圾邮件文件夹
- 在 Supabase Dashboard → Authentication → Users 中手动确认用户

**Q: 显示数据库错误？**
A: 确保已运行 `supabase-setup.sql` 脚本

**Q: 登录按钮没反应？**
A: 
- 打开浏览器控制台查看错误
- 确保 Site URL 配置正确
