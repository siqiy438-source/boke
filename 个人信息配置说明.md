# 个人信息配置说明

## 如何修改头像和个人简历

所有的个人信息都集中在 `constants.ts` 文件中的 `PERSONAL_INFO` 对象中。

### 位置

打开文件：`constants.ts`

找到 `PERSONAL_INFO` 对象（在文件开头部分）

### 可修改的内容

```typescript
export const PERSONAL_INFO: PersonalInfo = {
  // 1. 姓名 - 显示在"关于我"页面
  name: '我是思奇',
  
  // 2. 头像 - 修改这个URL来更换你的头像
  // 可以使用：
  // - 在线图片链接（如：https://example.com/avatar.jpg）
  // - 本地图片（需要放在 public 文件夹）
  avatarUrl: 'https://picsum.photos/400/400?grayscale',
  
  // 3. 副标题 - 显示在头像下方
  subtitle: '23岁 / 连续创业者 / 终身学习者',
  
  // 4. 自我介绍 - 可以有多段文字
  introduction: [
    '你好，欢迎来到我的精神角落。',
    '在这个信息爆炸的时代，我试图通过**写作**来对抗遗忘和焦虑。目前的我专注于实体店创业，同时对金融投资有着浓厚的兴趣。'
  ],
  
  // 5. 专栏介绍 - 可以添加、删除或修改
  columns: [
    {
      icon: 'Brain',  // 可选：Brain, Briefcase, TrendingUp, BookOpen
      title: '心智成长：',
      description: '探索潜意识、元认知与情绪力，向内求索。'
    },
    {
      icon: 'Briefcase',
      title: '商业思考：',
      description: '记录我在女装店实战中的经验、教训与价值重排方法论。'
    },
    {
      icon: 'TrendingUp',
      title: '财富逻辑：',
      description: '反脆弱投资笔记，建立属于年轻人的金融风控体系。'
    }
  ],
  
  // 6. 社交媒体链接
  socialLinks: {
    twitter: '#',   // 替换为你的 Twitter 链接，或删除这一行
    github: '#',    // 替换为你的 GitHub 链接，或删除这一行
    email: '#'      // 替换为你的邮箱链接（如：mailto:your@email.com），或删除这一行
  }
};
```

## 修改示例

### 1. 更换头像

```typescript
// 使用在线图片
avatarUrl: 'https://example.com/my-avatar.jpg',

// 或者使用本地图片（需要先把图片放到 public 文件夹）
avatarUrl: '/images/avatar.jpg',
```

### 2. 修改个人简介

```typescript
introduction: [
  '大家好，我是张三。',
  '我是一名全栈工程师，热爱技术，热爱生活。',
  '在这里记录我的学习和思考。'
],
```

### 3. 添加新的专栏

```typescript
columns: [
  {
    icon: 'BookOpen',
    title: '技术分享：',
    description: '分享编程技巧和最佳实践。'
  },
  // ... 其他专栏
],
```

### 4. 修改社交链接

```typescript
socialLinks: {
  twitter: 'https://twitter.com/yourusername',
  github: 'https://github.com/yourusername',
  email: 'mailto:your.email@example.com'
}
```

## 注意事项

1. 修改完成后，保存文件即可，页面会自动更新
2. 头像图片建议使用正方形，尺寸建议 400x400 像素以上
3. 如果使用本地图片，记得把图片文件放到项目的 `public` 文件夹中
4. 所有的文本都支持中文和英文
5. 修改时注意保持 JSON 格式的正确性（逗号、引号等）

## 常见问题

**Q: 修改后页面没有变化？**
A: 请检查是否保存了文件，并刷新浏览器页面（Ctrl+R 或 Cmd+R）

**Q: 如何使用自己的头像？**
A: 最简单的方法是使用在线图床上传图片后获取链接，或者将图片放到 `public` 文件夹中

**Q: 可以删除某些专栏吗？**
A: 可以！只需要在 `columns` 数组中删除对应的对象即可

**Q: 社交媒体链接可以不显示吗？**
A: 可以，删除对应的字段即可。例如不想显示 Twitter，就删除 `twitter: '#',` 这一行
